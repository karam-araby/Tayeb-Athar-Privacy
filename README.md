# Tayeb-Athar-Privacy

Privacy Policy page for Tayeb Athar Islamic App.

This repository contains the privacy policy for the Tayeb Athar mobile application, an Islamic app dedicated to helping users strengthen their faith and spiritual journey.

## Features

- Professional and user-friendly layout
- Islamic-themed design with appropriate styling

## Local Development

To view the privacy policy locally:

1. Clone the repository
2. Open `index.html` in your web browser
3. Or serve it using a local web server:
   ```bash
   python3 -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`
